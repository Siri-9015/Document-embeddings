# 🍽️ Interactive Recipe Card

An interactive and responsive recipe card built using **HTML, CSS, and JavaScript**. The project provides an engaging user interface where users can view recipe details, ingredients, cooking instructions, and interact with recipe features.

---

## ✨ Features

- 📖 Interactive recipe card interface
- 🍴 Displays recipe ingredients and cooking instructions
- ❤️ Favorite/Like functionality
- ⏱️ Cooking time and serving information
- 📱 Fully responsive design
- 🎨 Modern and clean UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Siri-9015/Interactive-Recipe-Card.git
```

### Navigate to the project

```bash
cd Interactive-Recipe-Card
```

### Run the project

Open the `index.html` file in your browser.
